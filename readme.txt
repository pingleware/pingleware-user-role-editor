A restored version from the https://wordpress.org/plugins/user-role-editor, which no longer permits adding new user roles

== Description ==
I have always used the https://wordpress.org/plugins/user-role-editor for quickly creating new custom user roles and changing the default user role, but I then notice on an update, the ability to add user roles has been remove and a new premium upgrade version is being promoted.

== Installation ==
1. Download the zip file from https://github.com/pingleware/pingleware-user-role-editor
2. Navigate to Plugins|Add New|Upload
3. Browse to the zip file and press install
4. Then activate.

Support multisite

== Frequently Asked Questions ==
To change the default role, just select the radio button of the new role, and the change is instant
